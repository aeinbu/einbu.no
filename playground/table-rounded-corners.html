Tables with rounded corners

It is hard to get rounded corners on tables when you have border or background styling of the cells.

The problem

1 Row 1, Cell 1 Row 1, Cell 2
2 Row 2, Cell 1 Row 2, Cell 2
3 Last Row, Cell 1 Last Row, Cell 2

When styling the table cells, the cells borders and background will overlap the rounded corners of the table.
This is because the table cells are rendered on top of the table element.

The solution

With column seperators

Table with header and footer
# Header 1 Header 2
# Footer 1 Footer 2
1 Row 1, Cell 1 Row 1, Cell 2
2 Row 2, Cell 1 Row 2, Cell 2
3 Last Row, Cell 1 Last Row, Cell 2
Table with header and no footer
# Header 1 Header 2
1 Row 1, Cell 1 Row 1, Cell 2
2 Row 2, Cell 1 Row 2, Cell 2
3 Last Row, Cell 1 Last Row, Cell 2
Table with no header or footer
1 Row 1, Cell 1 Row 1, Cell 2
2 Row 2, Cell 1 Row 2, Cell 2
3 Last Row, Cell 1 Last Row, Cell 2
